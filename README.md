# Milestone Planner (CSV 版)

CSV ファイルでマイルストーン（中長期計画）を管理・可視化するための、シンプルなブラウザツールです。

- ✅ インストール不要（`index.html` をブラウザで開くだけ）
- ✅ データはすべて手元の CSV で管理（サーバ送信なし）
- ✅ リスト表示とタイムライン表示で、計画を俯瞰

研究計画・教育計画・プロジェクト管理・個人のキャリアプランなど、汎用的に使えることを目指しています。

---

## 機能概要

- CSV ファイルの読み込み
  - 既存データの「置き換え」または「追加」が選択可能
- マイルストーン一覧（リスト表示）
  - レベル・カテゴリ・期間・状態・担当者などを一覧で確認
  - 行をクリックすると詳細表示
- タイムライン表示
  - `start_date`〜`end_date` を横軸にバー表示
  - カテゴリごとに俯瞰
- マイルストーンの編集・追加・削除
  - モーダルダイアログからタイトルや日付などを編集
- CSV 書き出し
  - 現在のマイルストーンを CSV としてダウンロード

---

## CSV フォーマット

ヘッダ行は以下の通りです。

```text
id,title,level,category,start_date,end_date,status,priority,owner,tags,notes
```
